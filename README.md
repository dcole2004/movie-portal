This project is a full-stack web application built using React for the frontend and PHP/MySQL for the backend. It allows users to submit movie reviews and view a live feed of recent entries. The primary goal was to implement a secure, optimized, and monitored networking environment.  The following enhancements were implemented:  Security Enhancements (Mandatory)SQL Injection (SQLi) Protection: Implemented using PHP Data Objects (PDO) with prepared statements in the backend to ensure all user input is parameterized before interacting with the database. For XSS protection I leveraged React’s built-in data binding, which automatically escapes strings to prevent malicious script injection. I created a dynamic system where reviews are stored in a MySQL database and retrieved via the Fetch API. I hosted the application on the Symptra server environment and cnfigured the production build to support subdirectory routing via a custom .htaccess file. To optomize performance we minified all production JavaScript and CSS through the npm run build process to reduce latency and improve page load speeds. Technology Used: React.js, PHP, MySQL, FileZilla (FTP), phpMyAdmin, VS Code.  Instructions: Clone the repository to your local machine.  Install dependencies using npm install.  Generate the production build using npm run build.  Upload the contents of the /build folder to the /htdocs/movieportal/ directory on the server.  Ensure db.php is located in the root of the /movieportal/ directory.  Import the provided .sql file into your MySQL database to initialize the reviews table.
